# PDF Generation Screen — Wireframe

This wireframe illustrates the PDF Generation Screen within the Resume Data Entry feature. This screen allows users to select a resume template and generate a PDF version of their resume.

```markdown
+--------------------------------------------------------------------------------------+
|                                 PDF GENERATION BAR                                   |
|--------------------------------------------------------------------------------------|
| Template: [Thumbnail Selector ▼]   [ Generate PDF ]   [ Download PDF ]  [ Back ]     |
+--------------------------------------------------------------------------------------+

+------------------------------------------------------+------------------------------+
|                 PDF PREVIEW (scroll)                 |  TEMPLATE OPTIONS (optional) |
|------------------------------------------------------|------------------------------|
|   +----------------------------------------------+   | Color Palette [● ● ● ●]     |
|   |                   Rendered PDF               |   | Font Family [Select ▼]      |
|   | (Actual visual output, high-resolution)      |   | Spacing        [Slider]     |
|   |                                              |   | Section Toggles [..]         |
|   +----------------------------------------------+   |                              |
+------------------------------------------------------+------------------------------+
```
